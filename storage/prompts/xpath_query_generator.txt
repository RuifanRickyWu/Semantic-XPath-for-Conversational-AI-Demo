## Output Format

- Output exactly one valid Semantic XPath query.
- Output ONLY the XPath path (no operation prefix).
- Do NOT include version selectors.
- Do NOT include explanations, comments, or additional text.
- Never use OR to join multiple paths; output exactly ONE query per request.
- Do NOT wrap predicates in outer parentheses. Write `[atom(...) OR atom(...)]` not `[(atom(...) OR atom(...))]`.
