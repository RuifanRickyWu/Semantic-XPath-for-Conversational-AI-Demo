## Query Syntax

### Path Navigation
- Paths use XPath-style navigation: /TodoList/Project/Task/SubTask
- Only use allowed node types: TodoList, Project, Task, SubTask
- This is a 4-level hierarchy: TodoList → Project → Task → SubTask

### Index Notation
- Positional index: Task[2] - select 2nd Task
- Negative index: Task[-1] - select last Task
- Range index: Task[1:3] - select Task 1 to 3 (inclusive)
- Global index: (/TodoList/Project/Task)[5] - select 5th Task globally

### Semantic Predicates

THREE types of semantic predicates (Paper Formalization):

1. **atom()** - Atomic predicate Atom(u, φ) on the node's OWN content
   - Use when the property can be determined from the node itself
   - Syntax: atom(content =~ "value")
   - Example: SubTask[atom(content =~ "testing")] - subtasks about testing

2. **agg_exists()** - Existential aggregation Agg∃ (max over children)
   - Use when checking if ANY child of a container has a property
   - Syntax: agg_exists(ChildType[atom(content =~ "value")])
   - Example: Project[agg_exists(Task[atom(content =~ "urgent")])] - projects with at least one urgent task

3. **agg_prev()** - Prevalence aggregation Aggprev (average over children)
   - Use when the property characterizes the container as a whole
   - Syntax: agg_prev(ChildType[atom(content =~ "value")])
   - Example: Project[agg_prev(Task[atom(content =~ "completed")])] - mostly completed projects

### Logical Operators
- AND: atom(content =~ "X") AND atom(content =~ "Y") - conjunction (product)
- OR: atom(content =~ "X") OR atom(content =~ "Y") - disjunction (max)
- Aggregation AND/OR: agg_exists(...) AND agg_exists(...), agg_exists(...) OR agg_prev(...)

## Decision Guide: When to Use Each Predicate

| Query Pattern | Predicate | Paper Formalization |
|--------------|-----------|---------------------|
| "urgent subtask" | atom() | Atom(u, φ) from attr(u) |
| "project with urgent tasks" | agg_exists() | Agg∃ = max over Sφ(u) |
| "completed project" | agg_prev() | Aggprev = avg over Sφ(u) |
| "active project" | agg_prev() | Aggprev = avg over Sφ(u) |
